### react-app
由于工程使用create-react-app快速开发的方式
需要安装node.js(安装步骤略)
- 安装create-react-app插件
```
cnpm install -g create-react-app
```
or
```
npm install -g create-react-app
```
- 进入项目根目录
```
cd react-app
```
- 第一次请先执行
```
cnpm install
```
- 启动
```
npm run dev
```
- 部署
```
npm run build 
```

插件名称 | 作用 
--- | ---
